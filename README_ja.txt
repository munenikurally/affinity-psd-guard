Affinity PSD Guard for Windows 11

概要

Affinity v3 で Photoshop の PSD ファイルを直接開くと、PSD内の一部レイヤーがずれて表示されることがあります。
このアプリは、その問題を回避するための Windows 11 用ツールです。

このツールは Affinity 本体の PSD 読み込み機能を置き換えるプラグインではありません。
Affinity は、PSD インポーターを差し替えるための公開プラグイン API を提供していないためです。

代わりに、このアプリは次の2つの機能を提供します。

1. PSD の問題点を解析する
2. Affinity v3 が誤解しやすいスマートオブジェクト情報を取り除いた PSD を生成する


起動方法

次のファイルをダブルクリックしてください。

Start-AffinityPsdGuard.cmd

別のフォルダへ移動する場合

アプリ本体は次の2つのファイルだけで動作します。

Start-AffinityPsdGuard.cmd
AffinityPsdGuard.ps1

別のディレクトリへ移動する場合は、この2つを同じフォルダに置いてください。
README_ja.txt と README_en.txt は説明書なので、動作には必須ではありません。


基本的な使い方

1. アプリを起動します。
2. Open PSD ボタンで PSD または PSB ファイルを選択します。
3. Analyze ボタンで PSD の解析結果を確認します。
4. Sanitize PSD ボタンを押すと、Affinity v3 向けに安全化した PSD を生成します。

PSD ファイルは、アプリのウィンドウへドラッグ＆ドロップすることもできます。


Sanitize PSD について

Sanitize PSD は、Affinity Photo 2 を起動せずに PSD を変換する機能です。

PSD のレイヤーには、表示用のピクセルデータとは別に、Photoshop のスマートオブジェクトや配置レイヤー用のメタデータが入っていることがあります。
Affinity v3 は、このメタデータを解釈するときにレイヤー位置をずらしてしまう場合があります。

このアプリは、PSD内の表示用ピクセルデータを残したまま、Affinity v3 が誤解しやすいスマートオブジェクト関連メタデータだけを取り除いた新しい PSD を作成します。

変換後の PSD は、自動的に Affinity v3 で開かれます。

生成されるファイル名は、標準では次のようになります。

元ファイル名.affinity-v3-safe.psd

同じ名前のファイルがすでに存在する場合は、上書きせず、日時付きの別名で保存します。


重要な注意

Sanitize PSD は元の PSD を直接上書きしません。
必ず別名の PSD を生成します。

この方式は、スマートオブジェクトを「編集可能なスマートオブジェクト」として保持することより、Affinity v3 でレイヤー位置を崩さずに開くことを優先します。

スマートオブジェクトの中身をPhotoshopのように再編集したい用途には向きません。
表示結果をAffinity v3で扱いやすくするための変換です。


今回のサンプルで分かったこと

提供された Peugeot306IIMaxi.psd を解析したところ、次の2つのレイヤーが Photoshop のスマートオブジェクトとして検出されました。

Balck
SK_Body

スマートオブジェクトは Photoshop 固有の変形情報を含むことがあり、Affinity v3 で直接開いたときに位置ずれの原因になる場合があります。

Sanitize PSD は、このスマートオブジェクト関連メタデータを削除した PSD を作成します。
今回のサンプルでは、変換後の Peugeot306IIMaxi.affinity-v3-safe.psd を再解析したところ、スマートオブジェクト由来の高リスク検出は0件になりました。


画面の項目

Open PSD
PSD または PSB ファイルを選択します。

Analyze
PSD のレイヤー情報を解析し、問題になりそうな要素を表示します。

Sanitize PSD
スマートオブジェクト関連メタデータを取り除いた Affinity v3 向け PSD を生成し、Affinity v3 で開きます。

Output
変換後の PSD ファイルの保存先です。

Save JSON
解析結果を JSON ファイルとして保存します。

制限事項

このアプリは元の PSD ファイルを直接書き換えません。

別名で新しい PSD を作成します。
変換後の PSD では、対象レイヤーはPhotoshopのスマートオブジェクトとしての編集情報を失う可能性があります。
ただし、表示用のレイヤーピクセルデータは残すため、Affinity v3で位置ずれなく開くことを優先できます。

このアプリの目的は、問題のある PSD を検出し、Affinity Photo 2 を起動せずに Affinity v3 向けの安全化PSDを作成することです。
